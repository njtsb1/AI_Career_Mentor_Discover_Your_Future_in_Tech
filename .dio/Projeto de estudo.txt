## Roteiro de Entrevista

Pergunta 1: O que é o DOM e como manipulá-lo?  
Resposta: É a representação da estrutura HTML em memória e pode ser manipulado via JavaScript.  

Pergunta 2: O que são props e state no React?  
Resposta: Props são dados passados de um componente pai para um filho; state é interno e gerenciado pelo componente.  

Pergunta 3: Como garantir performance em uma aplicação React?  
Resposta: Usar memoization, lazy loading, dividir código em componentes menores e evitar re-renderizações desnecessárias.  

Pergunta 4: O que é responsividade e como aplicá-la?  
Resposta: É adaptar a interface a diferentes tamanhos de tela usando CSS (media queries, Flexbox, Grid).  

Pergunta 5: Qual a diferença entre SPA e MPA?  
Resposta: SPA carrega uma única página e atualiza dinamicamente; MPA carrega várias páginas completas.
